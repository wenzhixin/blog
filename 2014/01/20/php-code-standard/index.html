通用规范

  • 代码缩进全部用 tab / 四个空格
  • SVN / Git 中新建文件编码类型统一用 utf-8 编码
  • 行宽：80 字符

PHP 代码划分

PHP 代码总是用完整的标准的 PHP 标签定界：

1
2
3
4
5

<?php
// 开头必须使用 <?php

// php文件必须以 ?> 结尾，并且保证其后面没有空格或空行。
?>

字符串文字

当字符串是文字（不包含变量），应当用单引号（apostrophe）来括起来：

1

$a = 'Example String';

操作符前后都要有空格

1
2
3

$c = $a + $b; // 操作符前后都要有空格

$company = 'Zend' . ' ' . 'Technologies'; // 字符串必需用 "." 操作符连接，在它的前后加上空格以提高可读性
