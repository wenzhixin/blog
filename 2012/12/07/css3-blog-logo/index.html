2、设置 logo 大小为 180px，设置所有的 div 为绝对定位

1
2
3
4
5
6

.wenyi-logo {
    width: 180px;
    position: relative;
}
.wenyi-logo div {
    position: absolute;
}

.wenyi-logo .border {
    width: 180px;
    height: 180px;
    background: #b1e4ff;
    border: 2px solid #789cb6;
    border-radius: 90px;
    box-shadow: 0 0 15px #fff;
}
