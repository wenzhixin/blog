## Ubuntu 12.04 安装 Nodejs

分类：操作系统 | 标签：Nodejs | 发布时间：2013-01-07 10:12:00

___

命令行：

	sudo apt-get install python-software-properties
	sudo add-apt-repository ppa:chris-lea/node.js
	sudo apt-get update
	sudo apt-get install nodejs npm
